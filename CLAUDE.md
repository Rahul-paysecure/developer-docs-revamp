# Memory

## Project

- Paysecure React developer documentation, deployed to Netlify from the GitHub `main` branch.

## Preferences

- When removing, adding, or resizing a UI element, reassess and correct the surrounding alignment, spacing, responsive layout, and visual balance before considering the update complete.
- For every project update: create a dedicated branch, commit and push the change, open a pull request, wait for GitHub and Netlify preview checks, merge it, and verify the production deployment.
